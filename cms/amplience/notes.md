# Amplience

Content management system

### Extensions

Simple web apps that extend the functionality of dynamic content. Tthere's 3 types of extensions:

1. Content field extensions: Extend the features of the content editing form
2. Content editor extensions: Augment the content editing experience and are displayed in a tab of the content form
3. Dashboard extensions: Full screen extensions displayed in a tab on the Dynamic Content Dashboard
